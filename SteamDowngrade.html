Steam Game Downgrader

About

Steam Game Downgrader is a user-friendly GUI tool designed to help gamers easily revert Steam games to previous versions. By utilizing the Steam API and steamcmd, it provides a seamless way to identify, download, and install specific game versions.

Features

  • Easy identification of game versions with Steam App IDs
  • Automatic retrieval and display of game logo and title
  • Guided process for locating Depot and Manifest IDs
  • Quick steamcmd setup if not already installed
  • Effortless game version downgrading

How to Use

Simply download or clone the repository, run main.py, and input the desired Steam App ID. Follow the intuitive on-screen instructions through confirmation of game title, Depot and Manifest ID input, and let the tool handle the rest.
